# Segmentation Report Structure

## 1. Executive summary

Segments count, optimal k, top business takeaway.

## 2. Data & features

Grain, proxies, feature justification table.

## 3. Methodology

K-Means rationale, k selection table (silhouette, min cluster %).

## 4. Limitations

No income, snapshot data, proxy balances.

## 5. Segment profiles

For each segment: size %, defining metrics, persona name, recommended actions.
